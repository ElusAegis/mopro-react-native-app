## Get started

1. Install dependencies

    ```bash
    npm install
    ```

2. Start the app

    ```bash
     npx expo start
    ```

    - start an android emulator

        ```bash
        npm run android
        ```

    - start an iOS simulator

        ```bash
        npm run ios
        ```

    - start a web app (WIP)

        ```bash
        npm run web
        ```
